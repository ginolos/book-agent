# Story Seed Template

Use this template when planning the 15 stories in a bedtime collection.

```markdown
# Story Seeds

## Story 1
- Name:
- Setting:
- Plot:
- Conflict:
- Resolution:
- Extra element:
- Lesson:

## Story 2
- Name:
- Setting:
- Plot:
- Conflict:
- Resolution:
- Extra element:
- Lesson:
```

## Uniqueness checks

Before drafting, review the full list and make sure:

- names do not repeat
- core plots do not repeat
- lessons do not cluster too narrowly
- settings vary enough to keep the collection fresh
- resolutions are not all solved in the same way
